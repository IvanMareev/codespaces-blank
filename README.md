# Галерея изображений

Веб-приложение для загрузки, просмотра и управления галереей изображений с хранением в S3-совместимом хранилище и метаданными в MySQL.

## Запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/IvanMareev/gallery_php.git
   cd gallery_php
   ```

2. Настройте переменные окружения:
   ```bash
   cp .env.example .env
   # Отредактируйте .env с вашими настройками
   ```

3. Запустите приложение:
   ```bash
   docker compose up --build
   ```

4. Откройте в браузере: `http://localhost:8000`
